# SQL Exercises Repository

## Overview

This repository contains a collection of SQL exercises covering fundamental and intermediate SQL concepts, including joins, unions, filtering, aggregates, grouping, and sorting. The exercises are organized into multiple sections to help practice different SQL operations and query techniques.

## Topics Covered

The exercises in this repository include:

* INNER JOIN
* LEFT JOIN
* RIGHT JOIN
* FULL OUTER JOIN
* Filtering with WHERE
* Sorting with ORDER BY
* Aggregations (SUM, AVG, COUNT)
* GROUP BY
* HAVING
* LIMIT
* Working with multiple tables (Employees, Projects)

## Exercise Structure

The exercises are grouped into:

* Exercise 1 – Basic SQL queries
* Exercise 2 – Filtering and sorting
* Exercise 3 – More advanced queries
* Exercise 4 – Joins, filtering, and aggregates

Each exercise includes:

* The SQL query
* The expected output table

## Purpose

The purpose of this repository is to practice SQL querying skills and understand how different SQL operations work together when working with relational databases.
